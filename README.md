## OpenCode Reliability Toolkit

Small tools that make AI coding agents more reliable in real engineering workflows.

| Tool | What it does |
|---|---|
| [opencode-waitfor](https://github.com/chncaesar/opencode-waitfor) | Adds `wait_for` for HTTP, TCP, and command readiness checks so agents stop writing fragile sleep loops. |
| [opencode-db-clean](https://github.com/chncaesar/opencode-db-clean) | Safely analyzes and vacuums bloated OpenCode SQLite databases with backups and integrity checks. |
| [opencode-session-reflection](https://github.com/chncaesar/opencode-session-reflection) | Reviews agent sessions as engineering evidence: communication gaps, recurring failures, and reusable workflow opportunities. |
| [opencode-fleet](https://github.com/chncaesar/opencode-fleet) | Lets one master OpenCode instance coordinate multiple remote OpenCode nodes across machines. |
